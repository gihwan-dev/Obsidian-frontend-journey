# Sentry 에러 모니터링

#resume-with-ai #장기 #인프라

## 요약
Sentry 무료 티어를 추가하여 서버 에러를 자동 포착하고 알림을 받는다.

## 왜?
- 현재 모든 에러가 `console.error`로만 출력
- 트래픽이 늘어나면 Vercel 로그를 수동으로 확인하는 것은 비실용적

## 어떻게?
- `@sentry/node` 설치 후 API 핸들러 에러 바운더리 래핑

## 예상 효과
- 프로덕션 에러 자동 감지
- 디버깅 시간 대폭 단축

## 참고
- 현재 일일 5-50명 수준에서는 과잉일 수 있음. 트래픽 증가 시 도입 권장.
