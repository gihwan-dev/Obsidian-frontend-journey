# 멀티 세션 & 자동 포커스

#claude-gui-app #단기 #세션관리 #UX

## 요약
여러 세션을 동시에 실행하고, AskQuestion 등 사용자 입력이 필요한 세션으로 자동 포커스를 이동하는 기능을 구현한다.

## 왜?
- 파워유저는 여러 작업을 동시에 실행해야 하는 경우가 많음
- bypass 모드에서도 AskQuestion 이벤트가 발생할 수 있으며, 이를 놓치면 세션이 멈춤
- 자동 포커스가 없으면 사용자가 수시로 모든 세션을 확인해야 하는 비효율 발생

## 어떻게?
1. 세션 매니저에서 활성 세션 목록 관리
2. JSON 스트림에서 `AskQuestion` 이벤트 감지
3. 감지 시:
   - 해당 세션 탭으로 자동 전환
   - macOS 시스템 알림 발송 (Notification Center)
   - 앱이 백그라운드면 Dock 아이콘 바운스
4. 세션 탭 UI: 탭/사이드바에 상태 뱃지(실행중/대기/입력필요) 표시
5. 키보드 단축키로 세션 간 빠른 전환

## 예상 효과
- 동시 다중 작업 효율 극대화
- 사용자 입력이 필요한 순간을 절대 놓치지 않음
- 세션 상태를 한눈에 파악 가능
