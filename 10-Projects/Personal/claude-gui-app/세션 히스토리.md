# 세션 히스토리

#claude-gui-app #장기 #세션관리 #UX

## 요약
과거 세션을 검색, 북마크, 태깅하고, 세션 결과 요약을 미리보기할 수 있는 히스토리 기능을 구현한다.

## 왜?
- 세션이 쌓이면 과거 작업을 찾기 어려워짐
- 성공적인 세션의 프롬프트/설정을 참고하여 재사용하고 싶을 때가 많음
- 프로젝트별, 시간별 작업 이력을 추적하는 것은 생산성 관리의 기본

## 어떻게?
1. **검색**:
   - 키워드 검색 (프롬프트, 결과 요약 내용)
   - 필터: 프로젝트, 날짜 범위, 상태(성공/실패), 모델
2. **북마크 & 태깅**:
   - 중요 세션 북마크(별표) 표시
   - 사용자 정의 태그 추가 (예: "리팩토링", "버그수정", "실험")
3. **결과 요약 미리보기**:
   - 세션 목록에서 compact 결과 미리보기
   - 변경 파일 수, 토큰 사용량, 소요 시간 요약
4. **데이터 저장**:
   - SQLite 또는 JSON 파일로 세션 메타데이터 영구 저장
   - compact 결과, 체크포인트 히스토리 포함

## 예상 효과
- 과거 작업의 빠른 검색 및 참조
- 성공 패턴 식별 및 재사용
- 장기적 생산성 추적
