# 스킬 사용 모니터링

#claude-gui-app #중기 #모니터링

## 요약
현재 세션에 로드된 스킬 목록을 표시하고, 실제 호출/사용 빈도를 추적하는 모니터링 기능을 구현한다.

## 왜?
- 어떤 스킬이 로드되어 있는지, 실제로 사용되고 있는지 가시성이 없음
- 사용하지 않는 스킬이 컨텍스트를 낭비하고 있을 수 있음
- 스킬 사용 통계를 보면 어떤 스킬이 유용한지 판단 가능

## 어떻게?
1. **로드된 스킬 목록**:
   - 세션 시작 시 로드된 스킬을 사이드바에 표시
   - 활성/비활성 상태 토글
2. **사용 추적**:
   - JSON 스트림에서 스킬 호출 이벤트 감지
   - 스킬별 호출 횟수, 마지막 사용 시간 기록
3. **통계 UI**:
   - 스킬별 사용 빈도 차트
   - 세션별/프로젝트별 스킬 사용 패턴
4. **추천**:
   - 미사용 스킬 비활성화 추천
   - 자주 사용하는 스킬 조합 프리셋 제안

## 예상 효과
- 스킬 사용의 가시성 확보
- 불필요한 스킬로 인한 컨텍스트 낭비 방지
- 스킬 활용 최적화를 위한 데이터 기반 의사결정
