# 워크플로우 저장 공유 & 템플릿

#claude-gui-app #중기 #워크플로우

## 요약
워크플로우를 JSON으로 직렬화하여 저장하고, 변수(플레이스홀더)를 지원하는 재사용 가능한 템플릿 시스템을 구현한다.

## 왜?
- 한 번 만든 워크플로우를 다른 프로젝트에서도 재사용할 수 있어야 함
- 팀 내 공유나 커뮤니티 공유를 위해 내보내기/가져오기 필요
- 변수 시스템이 있어야 범용 템플릿으로 활용 가능

## 어떻게?
1. **JSON 직렬화**:
   - 노드 위치, 연결, 설정 값을 모두 JSON으로 저장
   - 파일 확장자: `.workflow.json`
2. **템플릿 변수**:
   - `{{target_dir}}`, `{{feature_name}}`, `{{branch_name}}` 등
   - 워크플로우 실행 시 변수 값 입력 다이얼로그
3. **라이브러리**:
   - 개인 워크플로우 라이브러리 (로컬 저장)
   - 프로젝트별 워크플로우 (프로젝트 폴더에 저장)
4. **내보내기/가져오기**:
   - JSON 파일로 내보내기
   - 파일에서 가져오기 + 유효성 검증

## 예상 효과
- 워크플로우 재사용으로 반복 작업 시간 절약
- 팀/커뮤니티 간 베스트 프랙티스 공유
- 변수 시스템으로 유연한 템플릿 구성
