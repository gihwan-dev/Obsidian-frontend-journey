# 컨텍스트 윈도우 시각화

#claude-gui-app #중기 #모니터링 #UX

## 요약
현재 세션의 컨텍스트 윈도우 점유율을 시각적으로 표시하고, 파일별/요소별 비중과 overflow 경고를 제공한다.

## 왜?
- 컨텍스트 윈도우 초과 시 이전 맥락이 잘려나가 응답 품질 저하
- 현재 얼마나 컨텍스트를 사용하고 있는지 알 수 없으면 최적화 불가
- 어떤 파일/요소가 컨텍스트를 많이 차지하는지 알아야 효율적 관리 가능

## 어떻게?
1. **점유율 시각화**:
   - 프로그레스 바 또는 게이지 차트로 현재 점유율 표시
   - 색상 구분: 안전(초록) / 경고(노랑) / 위험(빨강)
2. **비중 분석**:
   - 파일별 토큰 점유율 표시
   - 시스템 프롬프트, 대화 히스토리, 도구 결과 등 요소별 비중
3. **Overflow 경고**:
   - 임계값(예: 80%) 도달 시 경고 알림
   - compact 실행 추천 또는 자동 compact 트리거
4. JSON 스트림에서 컨텍스트 관련 메타데이터 추출

## 예상 효과
- 컨텍스트 사용량의 실시간 가시성
- 품질 저하 전에 사전 대응 가능
- 어떤 요소를 정리해야 하는지 명확한 가이드 제공
