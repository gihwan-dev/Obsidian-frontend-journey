# 파일 트리 & 컨텍스트 관리 UI

#claude-gui-app #장기 #파일관리 #UI

## 요약
프로젝트의 파일 트리를 사이드바에 표시하고, 체크박스로 컨텍스트에 포함할 파일을 관리하며, CLAUDE.md를 인라인 편집할 수 있는 UI를 구현한다.

## 왜?
- Claude에게 어떤 파일을 컨텍스트로 제공하느냐에 따라 응답 품질이 크게 달라짐
- 현재 CLI에서는 파일 컨텍스트를 정밀하게 제어하기 어려움
- CLAUDE.md는 프로젝트별 지침의 핵심이지만 별도 에디터를 열어야 편집 가능

## 어떻게?
1. **파일 트리**:
   - 프로젝트 worktree의 파일 구조를 트리 형태로 표시
   - .gitignore 패턴 반영하여 불필요한 파일 숨김
   - 파일 클릭 → 미리보기 또는 에디터 열기
2. **컨텍스트 체크박스**:
   - 각 파일/폴더 옆에 체크박스 표시
   - 체크된 파일이 세션의 컨텍스트에 포함
   - 토큰 비용 미리보기 (각 파일의 예상 토큰 수 표시)
3. **CLAUDE.md 에디터**:
   - 프로젝트 루트의 CLAUDE.md를 인라인으로 편집
   - 마크다운 프리뷰 지원
   - 저장 시 즉시 반영

## 예상 효과
- 컨텍스트를 정밀하게 제어하여 응답 품질 향상
- 불필요한 파일 제외로 토큰 절약
- CLAUDE.md 편집의 편의성 향상
