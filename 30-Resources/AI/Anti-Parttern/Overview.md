# AI Anti-Pattern Overview

AI 코딩 에이전트가 반복적으로 보이는 한계를 실제 코드 사례 기반으로 정리한 문서.

## 목록

- [[잘못된 추상화]] — 도메인 경계가 아닌 학습 패턴 기반의 추상화 재생산
- [[과하고 무의미한 테스트]] — 행동 검증이 아닌 작업 완료 증빙으로서의 테스트 생성
- [[도메인 맥락에 대한 부재]] — 코드에 명시되지 않은 비즈니스 규칙 무시
- [[비대한 함수 입출력]] — 복잡성을 줄이지 않고 위치만 옮기는 구조 정리
- [[시각적 판단 불가]] — 시각적 결과를 확인하지 못해 발생하는 디자인 오류
