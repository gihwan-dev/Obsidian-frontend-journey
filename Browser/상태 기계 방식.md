상태 기계 또는 상태 기반 머신은 입력에 따라 상태가 변화하는 시스템을 의미한다.

1. 기본 개념
```
- 현재 상태가 있음
- 입력을 받음
- 입력에 따라 다음 상태로 전환
- 각 상태에서 특정 동작 수행
```

2. HTML 파서 예시
```text
입력: <div>Hello</div>

상태 변화 과정:
[데이터 상태] → '<' 발견 
↓
[태그 시작 상태] → 'd,i,v' 발견
↓
[태그 이름 상태] → '>' 발견
↓
[데이터 상태] → 'H,e,l,l,o' 처리
↓
[태그 종료 상태] → '</div>' 처리
```
