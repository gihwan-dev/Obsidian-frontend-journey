# [LG에너지솔루션] 인스턴스 통합 모니터링 대시보드

## 요약

LG 에너지솔루션 전용 인스턴스 통합 모니터링 대시보드. CPU, Lock, Long 데이터를 인스턴스별로 시각화하고, 임계치 기반 알람 및 상세 분석 기능을 제공.

## 전체 구조

- 화면 좌측 상단: CPU, Lock, Long 임계치 디스플레이
- 화면 우측 상단: Lock, Long 그리드 필터 설정 팝업 열기 버튼 및 임계치 설정 팝업 열기 버튼
- 화면 가운데: 각 인스턴스별 CPU, Lock, Long 데이터 시각화
- 화면 하단: 스탯별 TopN 차트
- 스플리터 드래그 앤 드롭으로 화면 하단 높이 수정 가능
- 즐겨찾기 토글 버튼으로 즐겨찾기된 인스턴스만 필터링 (카테고리 미적용)
- 카테고리 드롭다운: Location(다중 선택) / Service(다중 선택) 별 인스턴스 리스트 조회

## 기능 상세

### 임계치 설정 팝업

- 화면 우측 상단 톱니바퀴 버튼으로 열기
- 팝업 상단에 각 임계치에 대한 설명 표시
- 임계값 수정 후 저장 버튼 클릭으로 저장
- 유저별 영구 저장
- 즐겨찾기 수정 탭 포함

### 즐겨찾기 수정 팝업

- 테이블의 Favorite 칼럼의 별 아이콘 토글로 추가/제거
- Save 버튼으로 실제 저장 적용
- 필터, 검색, 정렬, 칼럼 리사이즈 기능 지원

### 그리드 필터 설정 팝업

- 화면 우측 상단 필터 아이콘 버튼으로 열기
- 팝업 상단에서 필터 적용 방식 설명 표시
- Lock, Long 탭별 개별 필터 적용 가능
- Add condition 버튼으로 필터 조건 추가
- Lock, Long 필터 별도 영구 저장
- Clear All 또는 각 행 우측 x 버튼으로 삭제

### 각 인스턴스 카드별 동작

- **Long 임계치 초과**: 카드 좌측 상단 붉은 아이콘 및 수치 표기
- **Lock 임계치 초과**: 카드 우측 상단 노란 아이콘 및 수치 표기
- **CPU 사용률 임계치 초과**: 차트 색상 붉은색으로 변경
- **CPU 미수집**: 차트 색상 회색으로 변경
- **알람 발생**: 해당 인스턴스 카드 배경 색상 변경 (알람 시스템 색상과 동일)
- **우클릭**: 연계 가능한 메뉴 노출
- **CPU 임계치 초과 시**: 두 개의 막대 차트 표시 (좌: 전날 CPU 평균치, 우: 현재 CPU값)

### 스탯 변경 팝업

- 화면 하단 TopN 영역의 제목 클릭으로 팝업 열기
- 스탯 클릭으로 체크박스 체크
- 단일 요소만 선택 가능
- 확인 버튼으로 적용 및 영구 저장

### 그리드 정렬

- 각 인스턴스 카드 드래그 앤 드롭으로 정렬 수정 가능

### 그리드 정렬 및 스플리터 프리셋 저장

- 화면 최상단 헤더의 우측 디스크 아이콘으로 프리셋 저장 팝업 열기
- 프리셋 제목 입력 및 저장 버튼으로 현재 그리드 레이아웃 저장
- 불러오기 버튼으로 저장된 레이아웃 불러오기
- 스플리터 위치도 함께 저장

### 인스턴스 디테일

- 3개의 차트 표시
- 현재 날짜 1시간 전부터 현재 시간까지의 5초 단위 값과 24시간 전의 5초 단위 값 표시
- 기본 Stat: CPU, Active Sessions, Memory Usage
- 드롭다운으로 스탯 변경 가능 (이미 표기 중인 스탯은 목록에서 제외)
- 인스턴스의 Active Session 목록 테이블 형태로 표시
- 헤더 클릭으로 정렬 (기본: Elapsed Time 기준 내림차순)
- 페이지당 데이터 갯수 수정 및 페이지 이동 가능
- 테이블 칼럼 순서/너비 변경 (유저별 영구 저장, 모든 인스턴스에 적용)
- OR/AND 기반 필터 적용 (유저별 영구 저장)
- 검색 및 칼럼 설정 초기화 기능

## 설정 기능

### 사용자 설정

- 사용자 추가/제거 등의 수정 가능

### 사용자별 화면 설정

- 특정 사용자가 어떤 화면을 볼 수 있는지 설정 가능

### 화면 세트별 서비스 설정

- 특정 화면에서 어떤 서비스 그룹들을 볼 수 있는지 설정 가능

### 서비스 설정

- 서비스 그룹에 어떤 인스턴스들이 포함될지 설정 가능

## 그리드 레이아웃 동작 방식

### 레이아웃 상태
- 모든 인스턴스의 위치 정보는 항상 메모리에 유지
- 화면에는 현재 조건에 맞는 인스턴스만 표시

### 필터링
- 필터로 인스턴스가 사라져도 위치 정보는 유지
- 필터 해제 시 원래 위치에 다시 나타남

### 드래그
- 카드를 드래그하면 해당 위치가 저장
- 숨겨진 카드의 위치는 영향받지 않음

### 신규 인스턴스
- 새 인스턴스는 기존 레이아웃 하단에 배치
- 기존 인스턴스의 위치는 변경되지 않음

### 화면 크기
- 화면이 좁아지면 넘치는 카드가 자동으로 재배치
- 겹치는 카드는 아래로 밀림

### 프리셋 저장
- 현재 레이아웃을 프리셋으로 저장 가능
- 스플리터 위치도 함께 저장

### 프리셋 로드
- 저장된 레이아웃을 불러오기 가능
- 화면이 너무 좁으면 로드 불가
- 저장 후 추가된 인스턴스는 하단에 배치
