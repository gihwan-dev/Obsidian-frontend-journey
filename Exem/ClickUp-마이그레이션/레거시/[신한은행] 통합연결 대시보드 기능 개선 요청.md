# [신한은행] 통합연결 대시보드 기능 개선 요청

## 요약

신한은행 통합연결 대시보드의 기능 개선 및 리팩토링 작업. Vue에서 React로의 마이그레이션 검토, API 변경 대응, UI/UX 개선 등을 포함.

## 상태 관련 참고

### MFO, MFT
- status: -1=RTS Disconnected, 0=Inactive, 1=Active, 3=Not Collected

### MFM, 추후 추가될 엑셈원 Postgresql
- status: 0=Inactive, 1=Active, 3=Not Collected

## 작업 리스트업

### 인스턴스 리스트 관련 수정 (담당: 원영)
- API 수정
- 칼럼 수정: IP, Port 제거하고 Host Name을 보여주도록 수정

### 알람 수정 작업 (담당: 영호)
- 서비스별 Status 대응
- Alerts 칼럼에서 인스턴스별 알람 갯수 카운트 제거
- DB Down, DM Down 관련 제거
- 연계 관련 수정
- 최근 알람 리스트 팝업 수정

## 피드백 사항

- 최댓치 99까지만 표시
- C | DB | DM | W 형태로 표시
- 반응형으로 텍스트 풀텍스트/축약 처리
- 카운트 값은 2자리 항상 고정
- 상단 토탈 알람 영역 참고
- 총 인스턴스 갯수 UI가 알람 영역 파악에 방해 → 위치 조정 필요

## React 마이그레이션 시 보존 기능

### 그리드 레이아웃 기능 (담당: 기환)
- 접었을때/펼쳤을때 UI 변화 보존
- 서비스별 인스턴스 리스트 표시

### 로그인/로그아웃 기능 (담당: 원영)
- 설정 페이지 이동 기능

### PA, RTM 연동 기능 (담당: 기환)
- `useNavigateToRTM` 훅 참고
- 연계 칼럼의 RTM, PA 버튼 클릭해 이동 가능

### 알람 기능 (담당: 영호)
- 화면 상단 중앙에서 통합 알람 표시
- 호버 시 알람 리스트 확인 가능
- DB Recent Alarm Popup
- Alerts 칼럼 버튼 클릭으로 진입
- 행 클릭 시 팝업 → 클리어 가능
- 각 서비스 그룹별 알람 현황 표시
- 각 서비스별 최고 레벨 알람 표시

## 제거할 기능

- 프리셋 저장
- 설정 > Service & Group 제거 (configuration에서 가능)
- 그리드 프레임 타이틀 클릭 팝업 제거
- minimize 기능

## UI 피드백

- 행 높이 줄이기
- 연계, Alerts, IP, Status 고정 너비로 수정
- 타이틀(Total) 형태로 수정하고 테이블 y축 위치 올리기

## 기술적 결정사항

### Vuetify 마이그레이션
- evui의 트리 그리드 컴포넌트 정렬이 동작 안함
- 드래그 앤 드롭 기반의 칼럼 정렬 변경이나 리사이즈 기능 불필요
- 정렬 로직 추가 및 아이콘 수정 방향으로 진행

### Status 표시
- DB, RTS 두개로 나누고 동그라미로 표현
- 파란색, 빨간색, PJS 색(노란색) 별도 필요
- 1,2,3 플래그로 (Oracle, Tibero 한정)
- 인스턴스 이름 / 비즈니스 이름 탭으로 전환 가능
