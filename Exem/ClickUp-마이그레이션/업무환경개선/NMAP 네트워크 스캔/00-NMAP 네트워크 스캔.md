# 🖨️ NMAP 네트워크 스캔

## 도입배경
- 레거시 업무 혹은 QS에서 이슈 확인 요청이 올 때, 테스트해볼 환경이 필요합니다
- 제품별, 버전별로 테스트 URL 이 다르고 주기적으로 달라지기도 하는데, 그때마다 팀원들이나 QS 에 물어봐야 하는게 불편합니다.

## 작업 내용
- 이 애플리케이션은 로컬 네트워크를 스캔하여 활성 호스트와 서비스를 모니터링하는 웹 기반 도구입니다. nmap을 사용한 네트워크 스캐닝과 Puppeteer를 이용한 웹 서비스 스크린샷 캡처 기능을 제공합니다.

- 박영호 대리님이 수집 BE 서버와 FE 작업을 해주셨고 https://nmap.fe1exem.xyz/ 에서 서비스 중입니다.

### 스캔 기능
- 사전 정의된 네트워크와 개별 호스트를 nmap으로 스캔
  - 발견된 호스트의 6000-10000 포트 범위를 TCP 스캔
- 스캔중인 네트워크 주소 목록:
  - 10.10.35.62/32, 10.10.47.0/24, 10.10.48.0/24

### 자동 스캔 스케줄링
- 서버 시작 시 초기 스캔 자동 실행
- 매일 00시(한국 시간 기준) 자동 재스캔 수행

### 사용 방법
- 대상 네트워크 범위를 정적으로 고정해두었는데, 필요시 추가 가능합니다
- 찾고자 하는 레거시 MFO, MFD, MFT, MFA 로 필터하면 바로 config 정보와 링크를 확인 할 수 있습니다
