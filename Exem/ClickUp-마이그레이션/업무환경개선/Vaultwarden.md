# Vaultwarden

- 계정: fe1@ex-em.com / exem1234!@#$

## 도입 배경
- 팀내 서비스가 너무 많아져서 로그인을 위해 ID / PW 를 공유하기 힘들어짐
- keycloak 이라는 도구로 SSO 를 도입했지만, OIDC 를 지원하지 않는 서비스의 경우 SSO 를 적용 할 수 없는 한계가 있음
- ID / PW 뿐 아니라 SSH 등 공용 계정에 대한 접근 방법을 쉽게 할 방법이 필요함
