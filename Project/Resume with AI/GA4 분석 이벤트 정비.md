# GA4 분석 이벤트 정비

#resume-with-ai #장기 #분석

## 요약
GA4에서 주요 사용자 행동을 추적하도록 커스텀 이벤트를 설정한다.

## 왜?
- 현재 모든 개선 방안은 사용자 행동에 대한 가정에 기반
- 실제 데이터가 있으면 향후 개선 방향을 검증 가능

## 어떻게?
- `gtag('event', ...)` 커스텀 이벤트 추가
- 추적 항목: 챗봇 사용률, PDF 다운로드율, 섹션별 스크롤 도달률, 평균 체류 시간

## 예상 효과
- 데이터 기반 의사결정
- "챗봇 사용률이 X%이다"라는 구체적 근거 확보
