# 접근성 기본 점검

#resume-with-ai #중기 #접근성

## 요약
키보드 네비게이션, 포커스 트래핑, 색상 대비 등 접근성 기본 사항을 점검한다.

## 왜?
- 기술적으로 인식 있는 채용담당자에게 접근성 미비는 부정적 신호
- 프론트엔드 개발자로서 접근성은 기본 역량

## 어떻게?
1. Lighthouse Accessibility 점수 측정
2. axe DevTools로 자동 검사
3. 키보드 네비게이션 완전성 확인
4. 챗봇 모달 포커스 트래핑
5. 색상 대비 비율 확인
6. 스크린 리더 테스트

## 예상 효과
- Lighthouse Accessibility 점수 향상
- 전문성 신호 강화
