# 컨텍스트 인식 이력서

#resume-with-ai #장기 #UX

## 요약
URL 쿼리 파라미터에 따라 강조하는 프로젝트와 스킬 순서를 조정한다.

## 왜?
- 다른 직무에 지원할 때 같은 이력서를 보여주는 대신, 해당 직무에 관련된 경험을 먼저 보여줄 수 있음
- "이 사람은 UX를 정말 이해하고 있다"는 인상

## 어떻게?
- `?role=frontend`, `?role=fullstack` 등 쿼리 파라미터 기반
- CSS `order` 속성으로 섹션/프로젝트 재배치, JS 최소화

## 예상 효과
- 채용 지원 시 맞춤형 이력서 URL 공유 가능
