# Claude 브랜드 테마 적용

#claude-gui-app #장기 #UI #디자인

## 요약
앱 전체에 Claude 브랜드 컬러(베이지/오렌지 톤)를 기반으로 한 일관된 디자인 테마를 적용한다.

## 왜?
- 일관된 브랜드 아이덴티티는 앱의 전문성과 완성도를 높임
- Claude 브랜드와의 시각적 일관성으로 사용자에게 친숙함 제공
- 기본 UI 프레임워크 스타일 그대로 사용하면 개성 없는 앱이 됨

## 어떻게?
1. **컬러 팔레트 정의**:
   - Primary: Claude 오렌지 (#D97706 계열)
   - Background: 베이지/크림 톤
   - Surface: 따뜻한 회색 계열
   - Accent: Claude 브라운/앰버
2. **컴포넌트 테마**:
   - 버튼, 입력 필드, 카드, 사이드바 등 모든 컴포넌트에 테마 적용
   - 다크 모드 지원 (향후)
3. **터미널 테마**:
   - xterm.js 커스텀 테마 (배경, 전경, ANSI 색상)
4. **아이콘 & 타이포그래피**:
   - Claude 스타일에 맞는 아이콘 세트
   - 깔끔한 산세리프 폰트 적용
5. CSS 변수 또는 Tailwind 커스텀 테마로 구현

## 예상 효과
- 전문적이고 완성도 높은 앱 외관
- Claude 브랜드와의 시각적 일관성
- 사용자에게 프리미엄한 사용 경험 제공
