# Git 기반 체크포인트 생성 롤백

#claude-gui-app #단기 #체크포인트 #Git

## 요약
세션 내에서 사용자가 원하는 시점에 git commit으로 체크포인트를 생성하고, 특정 체크포인트로 롤백할 수 있는 기능을 구현한다.

## 왜?
- Claude의 코드 변경이 마음에 들지 않을 때, 안전하게 되돌릴 수 있어야 함
- 체크포인트 없이는 "되돌리기"가 불가능하거나 매우 위험함
- bypass 모드에서 자유롭게 실행하되, 언제든 안전한 지점으로 복귀 가능해야 함

## 어떻게?
1. 체크포인트 생성:
   - 단축키 (예: Cmd+S) → worktree에서 `git add -A && git commit -m "checkpoint: {description}"`
   - 자동 체크포인트: 무한 루프 모드 각 사이클 완료 시
2. 체크포인트 목록 조회: `git log --oneline` 파싱 → 체크포인트 UI 표시
3. 롤백 실행:
   - 선택한 체크포인트로 `git reset --hard {commit_hash}`
   - 롤백 후 세션 재시작 옵션 제공
4. 안전장치: 롤백 전 현재 상태 자동 백업 체크포인트 생성

## 예상 효과
- 실험적 시도를 부담 없이 진행 가능
- "원하는 시점으로 되돌리기"로 작업 안정성 극대화
- 무한 루프 모드의 안전장치 역할
