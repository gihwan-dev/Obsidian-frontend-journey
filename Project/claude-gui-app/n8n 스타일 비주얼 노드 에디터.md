# n8n 스타일 비주얼 노드 에디터

#claude-gui-app #중기 #워크플로우 #UI

## 요약
워크플로우를 시각적으로 구성하는 n8n 스타일의 노드 기반 비주얼 에디터를 구현한다.

## 왜?
- 복잡한 다단계 작업을 코드 없이 시각적으로 구성할 수 있어야 함
- 노드 기반 에디터는 워크플로우의 흐름을 직관적으로 표현하는 최적의 방식
- 재사용 가능한 워크플로우 템플릿의 기반

## 어떻게?
1. **에디터 기반**: React Flow 또는 @xyflow/react 라이브러리 활용
2. **캔버스 기능**:
   - 노드 추가/삭제/이동
   - 노드 간 연결선(edge) 드래그로 생성
   - 줌/패닝
   - 미니맵
3. **노드 UI**:
   - 각 노드 타입별 아이콘 및 색상
   - 노드 클릭 → 상세 설정 패널 표시
   - 실행 상태 표시 (대기/실행중/완료/에러)
4. **실행 엔진**:
   - 노드 연결 그래프를 순회하며 순차 실행
   - 각 노드 완료 후 다음 노드로 컨텍스트 전달

## 예상 효과
- 비개발자도 이해할 수 있는 직관적 워크플로우 구성
- 복잡한 자동화를 시각적으로 설계·관리
- 워크플로우 공유/재사용의 기반
