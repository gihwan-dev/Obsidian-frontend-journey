## ⚠️ 미해결 논점

- **Worktree 정리 정책**: 세션 종료 시 자동 삭제 vs 수동 관리, 디스크 점유 문제
- **Context 전달 형식**: compact 결과의 포맷 및 주입 방식 표준화 필요
- **Tauri v2 안정성**: macOS 네이티브 기능(알림, 트레이 등) 지원 범위 확인 필요
- **xterm.js 성능**: WebView 내 터미널 렌더링 성능, 대량 출력 시 프레임 드롭 가능성
- **무한 루프 안전장치**: 비용 폭주 방지를 위한 hard limit 기준 (금액? 사이클 수? 토큰?)
- **워크플로우 복잡도 제한**: 노드 수 / 중첩 깊이 제한 필요 여부
