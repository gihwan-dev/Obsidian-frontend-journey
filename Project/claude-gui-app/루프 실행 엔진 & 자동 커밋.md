# 루프 실행 엔진 & 자동 커밋

#claude-gui-app #중기 #무한루프

## 요약
Claude Code 세션을 자동으로 반복 실행하며, 매 사이클 완료 시 자동으로 체크포인트(git commit)를 생성하는 무한 루프 엔진을 구현한다.

## 왜?
- 반복적 개선 작업(리팩토링, 버그 수정, 테스트 보강 등)을 자동화하려면 루프 실행이 필수
- 수동으로 "프롬프트 실행 → 확인 → 다시 실행"을 반복하는 것은 비효율적
- 매 사이클 자동 커밋으로 안전장치 확보 및 변경 추적 가능

## 어떻게?
1. 루프 실행 구조:
   ```
   [루프 시작] → 프롬프트 실행 → 자동 커밋 → 종료 판단 → 다음 사이클 or 종료
   ```
2. 사이클 관리:
   - 사이클 카운터 + 상한선 설정 (예: 최대 20사이클)
   - 사이클별 시작/종료 시간 기록
3. 자동 커밋:
   - 각 사이클 완료 시 `git add -A && git commit -m "loop cycle {n}: {summary}"`
   - 커밋 메시지에 사이클 번호와 변경 요약 포함
4. 상태 표시: 현재 사이클 번호, 경과 시간, 누적 토큰/비용

## 예상 효과
- 반복 작업 완전 자동화
- 사이클별 변경사항 추적 가능
- 사이클 상한선으로 비용 폭주 방지
