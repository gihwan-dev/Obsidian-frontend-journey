# 사이클 관리 기능

#claude-gui-app #중기 #무한루프 #UX

## 요약
무한 루프 모드의 사이클별 diff 요약, 일시정지/재개, 컨텍스트 누적 기능을 구현한다.

## 왜?
- 루프가 돌아가는 동안 "지금 뭘 하고 있는지" 알 수 있어야 함
- 중간 확인 후 방향을 조정하거나, 임시 중단이 필요한 상황이 있음
- 이전 사이클 결과를 다음 사이클에 반영해야 점진적 개선이 가능

## 어떻게?
1. **사이클별 diff 요약**:
   - 매 사이클 커밋 후 `git diff HEAD~1` 파싱
   - 변경 파일 수, ± 줄 수, 주요 변경 요약 표시
2. **일시정지/재개**:
   - Pause 버튼 → 현재 사이클 완료 후 대기
   - 대기 중 diff 확인, 프롬프트 수정 가능
   - Resume 버튼 → 다음 사이클 실행
3. **컨텍스트 누적**:
   - 이전 사이클의 Claude 응답을 compact(요약)
   - 요약본을 다음 사이클 프롬프트에 자동 주입
   - "이전 사이클에서 {요약}을 수행했습니다. 이어서 {프롬프트}"

## 예상 효과
- 루프 진행 상황의 실시간 가시성 확보
- 필요 시 즉시 개입 가능
- 사이클 간 문맥 유지로 점진적 개선 품질 향상
