# 설정 관리 GUI

#claude-gui-app #장기 #설정 #UI

## 요약
전역(Global) 및 프로젝트별(Local) 2단계 설정 구조를 GUI로 관리할 수 있는 설정 패널을 구현한다.

## 왜?
- 정책, MCP, 스킬, 모델 기본값 등 다양한 설정을 CLI나 파일 편집 없이 관리해야 함
- 전역 기본값 + 프로젝트별 오버라이드 구조로 유연한 설정 관리 필요
- GUI 기반 관리가 없으면 이 앱의 "간단한 관리" 가치가 반감

## 어떻게?
1. **2단계 구조**:
   - 전역 설정: 앱 설정 디렉토리에 저장 (`~/.claude-gui/settings.json`)
   - 로컬 설정: 프로젝트 루트에 저장 (`.claude-gui/settings.json`)
   - 로컬이 전역을 오버라이드
2. **관리 항목**:
   - 정책 관리: 금지 bash 커맨드 목록 편집, bypass 모드 토글
   - MCP 서버 관리: 설치된 MCP 목록, 활성화/비활성화 체크박스
   - 스킬 관리: 전역/로컬 스킬 추가/제거/편집
   - 모델 기본값: 기본 모델 선택 드롭다운
   - 알림 설정: 알림 종류, 사운드, 트레이 배지 토글
3. **UI 디자인**: 탭 또는 섹션 기반 설정 페이지

## 예상 효과
- 모든 설정을 한 곳에서 직관적으로 관리
- 프로젝트별 맞춤 환경 구성 용이
- CLI 설정 파일을 직접 편집할 필요 없음
