# 세션 완료 후 자동 PR 생성

#claude-gui-app #단기 #체크포인트 #Git #자동화

## 요약
세션 작업이 완료되면 worktree의 변경사항을 기반으로 GitHub/GitLab PR을 자동 생성하는 기능을 구현한다.

## 왜?
- 세션의 최종 목적은 결국 "메인 브랜치에 변경사항을 반영"하는 것
- 수동으로 worktree에서 메인으로 PR을 만드는 과정을 자동화하면 워크플로우가 완결됨
- Claude가 작업한 내용을 바탕으로 PR 제목/본문을 자동 생성하면 리뷰 효율도 향상

## 어떻게?
1. 세션 완료 시 "PR 생성" 버튼 또는 자동 트리거
2. PR 내용 자동 생성:
   - 세션 프롬프트 + compact 요약으로 PR 제목/본문 작성
   - 변경 파일 목록, 체크포인트 히스토리 포함
3. `gh pr create` 또는 GitHub API 호출로 PR 생성
4. 생성된 PR URL을 세션 결과에 표시
5. 옵션: PR 생성 후 worktree 자동 정리

## 예상 효과
- 세션 → PR → 리뷰 → 머지 워크플로우 자동화
- PR 작성 시간 절약
- 일관된 PR 포맷으로 리뷰 효율 향상
