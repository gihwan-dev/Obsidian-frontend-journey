# 세션별 설정 구현

#claude-gui-app #단기 #세션관리 #설정

## 요약
세션 생성 시 모델, MCP 서버, 스킬, 보안 정책을 개별 설정할 수 있는 시스템을 구현한다.

## 왜?
- 작업마다 필요한 모델, 도구, 정책이 다름 (예: 코드 리뷰는 Opus, 단순 작업은 Sonnet)
- MCP 서버도 작업 성격에 따라 다르게 선택해야 함
- 금지 커맨드 정책은 프로젝트/작업의 위험도에 따라 조절 필요

## 어떻게?
1. 설정 항목 데이터 모델 정의:
   - `model`: Opus | Sonnet | 기타
   - `mcp_servers`: 선택된 MCP 서버 목록
   - `skills`: 적용할 스킬 목록
   - `policies`: 금지 bash 커맨드, bypass 모드 설정
2. 세션 생성 다이얼로그에 설정 UI 통합
3. 전역 기본값 → 세션 오버라이드 구조 (2단계 설정)
4. 설정값을 CLI 실행 시 적절한 플래그/환경변수로 변환
5. 최근 사용한 설정 프리셋 저장 기능

## 예상 효과
- 작업별 최적화된 환경 구성 가능
- 불필요한 MCP/스킬 비활성화로 토큰 절약
- 보안 정책으로 위험한 명령어 실행 사전 차단
