# 알림 시스템 구현

#claude-gui-app #장기 #UX #시스템

## 요약
AskQuestion 발생, 작업 완료, 비용 상한 도달 등 주요 이벤트에 대한 macOS 시스템 알림 및 인앱 알림을 구현한다.

## 왜?
- 다중 세션 실행 시 앱이 백그라운드에 있을 수 있음
- 사용자 입력이 필요한 순간을 놓치면 세션이 불필요하게 대기
- 비용 상한 도달 시 즉시 알려야 비용 폭주 방지 가능

## 어떻게?
1. **이벤트별 알림**:
   | 이벤트 | 동작 |
   |--------|------|
   | AskQuestion 발생 | 해당 세션 자동 포커스 + 시스템 알림 |
   | 다중 세션 중 입력 필요 | 자동 세션 전환 |
   | 장시간 작업 완료 | macOS 알림 센터 + 사운드 |
   | 비용 상한 도달 | 경고 알림 + 일시정지 |
2. **macOS 연동**:
   - Tauri notification API 사용
   - Dock 아이콘 배지/바운스
   - 알림 센터 통합
3. **인앱 알림**:
   - 토스트 알림 (우상단)
   - 알림 히스토리 목록
4. **설정**: 알림 종류별 활성화/비활성화, 사운드 선택

## 예상 효과
- 중요 이벤트를 절대 놓치지 않음
- 백그라운드 작업 시에도 안심
- 사용자가 원하는 수준으로 알림 커스텀 가능
