# 체크포인트 Diff & 타임라인 뷰

#claude-gui-app #단기 #체크포인트 #UI

## 요약
체크포인트 간 코드 변경사항을 diff로 시각화하고, 체크포인트 히스토리를 타임라인 UI로 표시한다.

## 왜?
- 체크포인트 목록만으로는 "이 시점에 무엇이 바뀌었는지" 알 수 없음
- diff 시각화가 있어야 롤백 시점을 정확하게 판단 가능
- 타임라인 뷰로 작업 흐름의 전체 진행 상황을 한눈에 파악

## 어떻게?
1. Diff 표시:
   - `git diff {commit_a} {commit_b}` 결과를 파싱
   - 프론트엔드에서 diff 뷰어 컴포넌트 구현 (monaco diff editor 또는 커스텀)
   - 파일별 변경 목록 + 인라인 diff
2. 타임라인 뷰:
   - 세로 타임라인 UI (체크포인트 = 노드)
   - 각 노드에 시간, 설명, 변경 파일 수, ± 줄 수 표시
   - 노드 클릭 → diff 상세보기
3. 체크포인트 간 비교: 임의의 두 체크포인트 선택 → 비교

## 예상 효과
- 코드 변경 이력을 시각적으로 추적
- 롤백 결정을 위한 정보 제공
- 작업 진행 상황의 직관적 파악
