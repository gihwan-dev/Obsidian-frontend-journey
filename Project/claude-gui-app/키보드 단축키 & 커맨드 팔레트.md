# 키보드 단축키 & 커맨드 팔레트

#claude-gui-app #장기 #UX

## 요약
모든 주요 기능에 키보드 단축키를 매핑하고, Cmd+K로 접근하는 커맨드 팔레트를 구현한다.

## 왜?
- 파워유저에게 키보드 중심 조작은 생산성의 핵심
- 마우스 클릭 대비 단축키가 평균 3~5배 빠름
- 커맨드 팔레트는 모든 기능에 빠르게 접근하는 허브 역할

## 어떻게?
1. **단축키 매핑**:
   - 체크포인트 생성: Cmd+S
   - 새 세션: Cmd+N
   - 세션 전환: Cmd+1~9
   - 루프 시작/정지: Cmd+L
   - 사이드바 토글: Cmd+B
2. **커맨드 팔레트 (Cmd+K)**:
   - 검색 가능한 명령어 목록
   - Fuzzy 검색 지원
   - 최근 사용 명령어 상단 배치
   - 키보드로 전체 탐색 가능
3. **커스텀 단축키**:
   - 사용자 정의 단축키 설정
   - 충돌 감지 및 경고

## 예상 효과
- 파워유저의 생산성 극대화
- 모든 기능에 2초 이내 접근 가능
- 직관적인 조작 경험
