# 프로젝트 기반 세션 생성 관리 UI

#claude-gui-app #단기 #세션관리 #UI

## 요약
Git 저장소 단위의 프로젝트를 등록하고, 프로젝트 내에서 세션을 생성/관리/삭제하는 UI를 구현한다.

## 왜?
- 모든 작업은 "프로젝트 단위"로 관리되는 것이 스펙의 핵심 원칙
- 세션은 프로젝트 안에서 생성되어 worktree로 격리됨
- 사용자가 어떤 프로젝트에서 어떤 세션이 돌고 있는지 한눈에 파악해야 함

## 어떻게?
1. 프로젝트 등록: 로컬 Git 저장소 폴더 선택 → 프로젝트 목록에 추가
2. 세션 생성 다이얼로그:
   - 프롬프트 입력
   - 모델 선택 (Opus / Sonnet 등)
   - MCP 서버 체크박스 선택
   - 스킬 선택
   - 정책 설정 (금지 bash 커맨드 등)
3. 세션 목록 UI: 상태(실행중/대기/완료), 프로젝트명, 시작 시간 표시
4. 세션 컨트롤: 시작, 일시정지, 재개, 종료, 삭제
5. 상태 관리: zustand 또는 jotai로 세션 상태 전역 관리

## 예상 효과
- 사용자가 직관적으로 작업을 생성·관리 가능
- 동시에 여러 프로젝트의 여러 세션을 한 곳에서 모니터링
- 세션 설정의 유연성으로 다양한 작업 시나리오 지원
